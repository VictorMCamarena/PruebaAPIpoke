# PokéApp

Una aplicación SPA hecha con React y TailwindCSS que muestra una lista de Pokémon obtenida desde la API pública [PokeAPI](https://pokeapi.co/). Permite explorar Pokémon aleatorios y practicar conceptos como rutas, componentes y estilos responsivos.

---

## 🚀 Tecnologías utilizadas

- React
- TailwindCSS
- React Router DOM
- PokeAPI (https://pokeapi.co)

---

## 🛠️ Requisitos previos

Asegúrate de tener instalado:

- [Node.js](https://nodejs.org/) (versión recomendada: 18.x o superior)
- npm o yarn

---

## 📦 Instalación

1. **Clona el repositorio:**

```bash
git clone https://github.com/tu-usuario/pokeapp.git
cd pokeapp
```

2. **Instala las dependencias:**

```bash
npm install
```

3. **Inicia la aplicación:**

```bash
npm run dev
```

Esto abrirá la app en tu navegador en [http://localhost:5173](http://localhost:5173) (si usas Vite).

---

## 🌐 API utilizada

Los datos se obtienen desde [https://pokeapi.co](https://pokeapi.co), una API REST pública que proporciona información detallada sobre los Pokémon.
